Для использования скрипта нужно написать

python3 shikaku.py -s [file with puzzle]
Решает головоломку из файла file_with_puzzle и выводит ответ в консоль

python3 shikaku.py -g [number_of_solutions] [out_file_solution]
Генерирует головоломку с number_of_solutions решениями и выводит ответ в файл out_file_solution
